Step 1: Yupoo相册 - 提取图片URL
1
加载Yupoo Cookie
从 logs/yupoo_cookies.json 加载
2
访问相册页面
https://x.yupoo.com/gallery/{album_id}
3
CDP拦截提取图片URL
拦截 /api/albums/{id}/photos 获取含hash完整路径
pic.yupoo.com/{user}/{photo_id}/{hash}.jpg
4
输出：图片URL列表
最多14张图（第15张预留给尺码表）
image_urls = [ "https://pic.yupoo.com/lol2024/xxx/abc.jpg", "https://pic.yupoo.com/lol2024/xxx/def.jpg", ... ]
Step 2: Google图片搜索 - 用图片搜索官网
1
技术方案选择
Playwright 最稳定 - 模拟真实浏览器
SerpAPI 付费API，稳定但需API Key
Python requests 直接调用，但Google反爬严格
2
Playwright方案（推荐）
# 1. 访问 Google Images await page.goto("https://www.google.com/imghp") # 2. 点击图片搜索按钮 await page.click("button[aria-label='Search by image']") # 3. 上传图片文件 file_input = page.locator("input[type='file']#searchboxinput") await file_input.set_input_files(local_image_path) # 4. 等待搜索结果 await page.wait_for_load_state("networkidle") # 5. 获取官网链接（通常第一个结果） official_site = await page.evaluate(""" () => { // 查找第一个可信来源（官网特征） const links = document.querySelectorAll('a'); for (const link of links) { const href = link.href; const text = link.innerText; // 过滤品牌官网特征 if (href && !href.includes('google') && !href.includes('cache')) { return { url: href, title: text }; } } return null; } """)
3
提取官网标题
# 从搜索结果提取官网名称 # Google图片搜索会显示 "查看官网" 按钮 official_title = await page.evaluate(""" () => { // 查找包含 "官网" 或 "official site" 的元素 const elements = document.querySelectorAll('*[data-ved], [href*="url="]'); // 提取第一个非Google链接的标题 } """)
4
验证URL来源
🔗 验证来源URL（必须记录）
https://www.nike.com/xxx https://www.adidas.com/xxx https://www.supreme.com/xxx

用于后续验证和追溯，确保产品标题来源可信

Step 3: Excel模板填充
📊 必须填充的字段
B列商品名称← 官网英文标题
E列商品主图← Yupoo第1张图URL
F列详情图片← Yupoo第2-14张图URL
I列上架状态固定值 Y
J列运费模板固定值 "默认模板"
P列商品重量固定值 0.512
AC列售价需确定（默认值99）
🔗 来源验证字段（建议填充）
N列商品备注可填搜索来源URL
T列SEO标题基于官网标题生成
V列SEO关键词提取自官网
