Step 1: Yupoo相册数据提取
1.1 提取图片外链
通过XHR拦截获取完整的图片URL（含Hash）
# CDP拦截: /api/albums/{id}/photos # 提取: http://pic.yupoo.com/lol2024/{photo_id}/{hash}.jpeg # 限制: 最多14张图片URL
1.2 提取元数据
商品标题相册标题（去除内部编号如H110）
副标题中文翻译/描述
商品描述HTML格式的品牌+款号+尺码信息
属性(H)品牌|xxx\n系列|xxx\n款号|xxx\n颜色|xxx\n尺码|xxx
类别(K)男装,外套 等
标签(L)DESCENTE,ALLTERRAIN 等
重量(P)单位kg, 3位小数
关键约束: 图片URL
1. 必须使用 pic.yupoo.com 外链
2. 多个URL用换行分隔
3. 首图(E列)单独一列, 其他图片(F列)
Step 2: Excel模板填充
2.1 加载Excel模板
模板文件: 商品导入模板.xlsx
包含2个Sheet: 商品信息 + 计量单位
import openpyxl wb = openpyxl.load_workbook('商品导入模板.xlsx') ws = wb['商品信息'] # 第3行开始填数据
2.2 填充规则（33列完整字段）
列字段名填写规则示例
A商品ID空=新增商品（留空）
B商品标题*最多255字符DESCENTE ALLTERRAIN BS W ZIP JACKET
C副标题最多255字符DESCENTE 联名系列 男子防风防水夹克
D商品描述HTML代码<p>品牌链接</p><p>描述...</p>
E商品首图*单URLhttp://pic.yupoo.com/.../xxx.jpeg
F商品其他图片多个URL用换行http://pic.yupoo.com/...\nhttp://pic.yupoo.com/...
H属性属性名|属性值品牌|DESCENT\n系列|ALLTERRAIN\n款号|22-0975-91
I商品上架*Y=上架 N=下架N（必须下架！）
J物流模板*系统配置的模板默认模板
K类别名称英文逗号隔开男装,外套
L标签英文逗号隔开DESCENTE,ALLTERRAIN,防水夹克
M计量单位件/个 等件
N商品备注最多50字相册ID:232338513 | 款号:22-0975-91
O不记库存*Y=不记 N=记N
P商品重量*kg, 3位小数0.8
X规格1Color+尺码表Color\nBlack
Y规格2Size+尺码详情Size\nM: 肩宽46cm/胸围116cm...
ABSKU值Color:xxx\nSize:xxxColor:Black\nSize:M
ACSKU图片完整URLhttp://pic.yupoo.com/...
AD售价*2位小数88.99
AE原价2位小数149.99
AF库存最多9位整数100
强制规则: 必须设为下架状态
I列(商品上架) 必须填写 N！
这是业务红线: 禁止自动上架,必须人工审核后再上架
