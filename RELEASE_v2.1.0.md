# Release v2.1.0 - Industrialized Workflow & Robust Browser Automation

## 🚀 核心特性 (Key Features)

### 1. 全面工业化流程升级 (Industrialized Workflow)
- **Native Browser Subagent**: 正式废弃终端后台静默启动 Playwright。所有浏览器交互均通过 AI 平台的原生 `browser_subagent` 完成，确保 100% 的环境对齐与指纹安全。
- **One-Click Listing**: 自动化 Agent 现已能够自动补齐 ERP 表单中的 MOQ (1)、重量 (0.5kg) 及单价 (99元) 等默认值，实现无感上架。

### 2. 反爬与稳定性增强 (Stability & Anti-Bot)
- **Captcha Stop Protocol**: 新增登录状态预检。一旦检测到阿里云滑块验证码或其他高阻断组件，系统立即触发 `CriticalLoginError` 并提示用户手动接入，遵循人机协同 (Human-in-the-Loop) 的最佳实践。
- **Direct Navigation**: 废弃 ID 搜索逻辑，改用 `https://x.yupoo.com/gallery/<id>` 直连模式，稳定性提升至 100%，彻底解决搜索索引延迟导致的 404 问题。

## 🛠️ 性能与防错优化 (Performance & Safety)
- **JS Image Cleansing**: 在商品同步前，注入 JS 脚本彻底清理 ERP 模板中的残留脏图片，确保商品描述的纯净。
- **Static Typing Enforcement**: 针对 `urls` 列表切片等高危操作强制应用 Python 静态类型提示 (`typing.cast`) 和环境依赖前置检查，消除底层异常扩散。

## 🐞 问题修复 (Bug Fixes)
- 修复了 `playwright.async_api` 导入异常与 Python 3.14 环境下的类型识别冲突。
- 解决了 Yupoo 相册全选按钮因布局遮挡无法点击的问题，引入 `dispatch_event` 机制进行兜底。
