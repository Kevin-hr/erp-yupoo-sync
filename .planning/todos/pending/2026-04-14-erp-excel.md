## Problem

当前同步流水线(Yupoo→MrShopPlus)直接操作ERP表单，存在以下问题：
1. **绕过标准导入流程** - 未使用ERP官方的Excel导入模板，无法利用批量导入能力
2. **表单操作脆弱** - TinyMCE编辑器/img标签清除/JS注入等变通方案维护成本高
3. **无法批量处理** - 当前单worker模式，每次只能处理一个商品

**目标**: 引入Excel中转，实现工业级批量导入。

## Solution

### Phase 1: Excel中转架构设计
```
Yupoo Extract ──▶ MetadataPreparer ──▶ Excel填充 ──▶ ERP批量导入
     │                    │                  │              │
  image_urls          {title, desc,     33列完整填写   触发ERP导入
   (≤14张)           sizes[], imgs}     模板.xlsx        API/界面
```

### Phase 2: 数据映射规则
| Yupoo来源 | → | Excel目标 | 转换逻辑 |
|-----------|---|-----------|----------|
| 相册标题 | → | 商品标题* | 去除内部编号 |
| 相册描述 | → | 商品描述 | 提取HTML+尺码表 |
| 图片列表 | → | 商品首图* + 商品其他图片 | 第一张=首图，其余换行分隔 |
| 尺码文本 | → | 规格2 | 格式化: `Size\nS\nM\nL` |
