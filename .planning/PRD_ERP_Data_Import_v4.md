## 1. 项目概述与目标

### 1.1 业务背景

| 维度 | 现状 |
|------|------|
| **上游数据源** | Yupoo 相册（4000+款商品，图片+标题+描述） |
| **目标系统** | MrShopPlus ERP（批量导入模式） |
| **历史方案** | Playwright UI 自动化（约2分钟/单，脆弱的TinyMCE/JS注入） |
| **新方案** | **Excel中转批量导入**（工业级稳定） |

### 1.2 核心问题

| 问题 | 原因 | 后果 |
|------|------|------|
| 表单操作脆弱 | TinyMCE iframe操作/img标签清除/JS注入绕过maxlength | 维护成本高，ERP UI更新即失效 |
| 无法批量处理 | 直接表单操作每次只能处理1个商品 | 4000+商品无法规模化 |
| SPA路由踩踏 | ERP"复制"是Vue路由跳转非新Tab | 所有并发方案均失败 |

### 1.3 新方案优势

| 对比项 | 旧方案(表单操作) | 新方案(Excel中转) |
|--------|-----------------|------------------|
| 稳定性 | ⚠️ 依赖CSS选择器 | ✅ 依赖结构化数据 |
| 批量能力 | ❌ 单商品 | ✅ 批量导入 |
| 维护成本 | 高(TinyMCE/JS注入) | 低(Excel字段映射) |
| 人工干预 | 每步需监控 | 仅最终导入点击 |
| 错误恢复 | 断点续传复杂 | Excel重新填充 |

### 3.2 TLD流程节点

| 节点 | Trigger | Logic | Deliver |
|------|---------|-------|---------|
| **EXTRACT** | album_id | XHR拦截/api/albums/{id}/photos | image_urls[], title, desc, sizes[] |
| **EXCEL_PREPARE** | image_urls[], metadata | 填充34列Excel模板 | `导入模板_{album_id}.xlsx` |
| **ERP_IMPORT** | Excel文件 | 打开ERP导入界面，上传文件 | 商品上架完成 |

## 4. 数据映射规则

### 4.1 Yupoo → Excel字段映射

| Yupoo来源 | → | Excel目标 | 转换逻辑 |
|-----------|---|-----------|----------|
| 相册标题 | → | `商品标题*` (B) | 去除内部编号(如H110)，保留品牌+品名 |
| 完整HTML描述 | → | `商品描述` (D) | 保留完整HTML，不做清洗 |
| 第1张图片 | → | `商品首图*` (E) | 直接复制URL |
| 第2-N张图片 | → | `商品其他图片` (F) | 换行分隔，最多13个 |
| (固定值) | → | `商品上架*` (I) | `N` |
| (固定值) | → | `物流模板*` (J) | `Clothing` |
| 相册分类 | → | `类别名称` (K) | 映射表: BAPE→男装, Nike→鞋类 |
| 相册标题 | → | `标签` (L) | 直接复制标题 |
| (固定值) | → | `不记库存*` (O) | `Y` |
| (固定值) | → | `商品重量*` (P) | `0.8` |
| 描述中的尺码行 | → | `规格2` (Y) | 格式化为 `Size\nS\nM\nL` |
| (固定值) | → | `售价*` (AD) | `88.99` |
| (固定值) | → | `原价` (AE) | `149.99` |
| (固定值) | → | `库存` (AF) | `100` |

### 4.2 尺码提取逻辑

```python
# 从Yupoo描述中提取尺码
def extract_sizes(description: str) -> str:
    """
    输入: "S M L XL 2XL 3XL"
    输出: "Size\nS\nM\nL\nXL\n2XL\n3XL"
    """
    sizes = description.strip().split()
    # 过滤非尺码内容
    valid_sizes = [s for s in sizes if re.match(r'^(S|M|L|XL|2XL|3XL|XS|XXL)$', s, re.I)]
    return "Size\n" + "\n".join(valid_sizes)
```
