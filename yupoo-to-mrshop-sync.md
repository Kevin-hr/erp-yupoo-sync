# Yupoo to MrShopPlus Synchronization Flow (SOP)

// turbo-all

1. Navigate Yupoo: `All Categories` -> `BAPE-芭比`.
2. Iterate products: Left-to-right, Top-to-bottom.
3. Detail: Enter product detail -> Copy URL.
4. Backend: Click `进入后台` -> Search product name.
5. Links: Select all (≤14) -> `批量外链` -> `链接` -> `复制`.
6. ERP Sync: Enter ERP and upload images.
