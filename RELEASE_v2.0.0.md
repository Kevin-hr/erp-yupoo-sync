# Release v2.0.0 - 工业级跨平台自动化流水线升级

本次更新完成了 ERP 数据同步流水线的全面工业化改造，引入了全新的决策认知系统 (Decision System)，从底层重构了任务编排与容错机制，大幅提升了图片批量上传、产品上架以及状态同步的稳定性和全自动化流程覆盖率。

## ✨ 新特性 (New Features)
* **工业级自动化决策系统**：引入完整的 `decision_system` 模块（含任务路由 `router.py`、断路器 `circuit_breaker.py` 等），基于 5W1H/MECE 原则重构高并发任务调度引擎。
* **原生防风控框架 (BROWSER SOP)**：实装 `BROWSER_SUBAGENT_SOP.md` 的隔离规范，在处理如 Yupoo 等严格反爬业务目标时，杜绝 429 和账号拦截等异常。
* **富文本描述自动化处理**：在 `sync_pipeline.py` 中无缝集成第一行商品描述格式化逻辑，一键自动植入符合规范的品牌超链接。

## ⚡️ 性能与优化 (Optimizations)
* **健壮性与速率控制重构**：结合路由网关策略优化了跨平台交互与上传，降低因网络抖动导致的“同步中”卡顿报错。
* **模块化系统瘦身**：淘汰清理了传统落后的离散型脚手架（如 `erp_image_uploader.py`, `mrshop_image_upload.py`），整合出极简、高内聚的工作流框架，显著减低内存与计算损耗。

## 🐞 问题修复 (Bug Fixes)
* 解决了最新环境（含 Python 3.14/Pyre2）下的 Playwright 类型检查与 Import 阻断卡点，使得整条流水线跑通“最后一公里”。
* 修正了早前因脚本耦合过强可能产生的前置业务脏乱风险。

## ⚠️ 破坏性更新 (Breaking Changes)
* **核心架构升级**：完全废除了以前的独立入口模块。如今所有调用需统一收口至基于 `decision_system` 的流程控制。老用户请务必对照最新流程（如 `erp_yupoo_sync_v2.md`）重新配置你的执行通道。
